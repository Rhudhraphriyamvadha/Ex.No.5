# INTRODUCTION

Prompt engineering is an important technique used to improve interactions with AI systems. The structure and clarity of prompts greatly influence the quality of responses generated by AI models.
This experiment compares two prompt styles:
1. Naïve Prompt
2. Basic Prompt

---

# PROMPT TYPES

## 1. Naïve Prompt

A naïve prompt is broad, simple, or unstructured with minimal instructions.

### Example:

“Tell me about Artificial Intelligence.”

---

## 2. Basic Prompt

A basic prompt is clear, detailed, and structured with proper instructions and context.

### Example:

“Explain Artificial Intelligence in simple terms for undergraduate students with two real-world examples.”

---

# ANALYSIS

## Impact of Prompt Clarity

The experiment showed that prompt clarity significantly improves AI-generated responses.

### Findings:

* Basic prompts generated more accurate and detailed outputs.
* Structured instructions improved contextual understanding.
* Adding audience type and examples increased relevance.
* Naïve prompts worked adequately for simple tasks but lacked depth.

---


## RESULT

The experiment successfully compared naïve prompts and basic prompts using ChatGPT.


## CONCLUSION

This experiment demonstrated that prompt engineering plays a major role in improving AI response quality. Basic prompts provide clearer instructions and context, enabling ChatGPT to generate more accurate, detailed, and meaningful outputs.

The study concludes that:

* Structured prompts are more effective than broad prompts.
* Prompt clarity directly impacts response quality.
* Well-designed prompts improve educational and professional AI applications.
